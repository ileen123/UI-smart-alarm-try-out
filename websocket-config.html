🧪 Test Mode & Message Monitoring

Control how WebSocket messages are handled and monitored. Test mode generates ping messages only, while real mode attempts actual connections but can still show ping messages for monitoring.

💡 Modes:
• Test Mode: No real WebSocket connections, ping messages only
• Real Mode: Attempts WebSocket connections to localhost:8080
• Ping Messages: Visual monitoring of all outbound message attempts
Add ?testMode=true or ?showPings=false to URLs for automatic configuration.
